Abstract:

O Extensible Stylesheet Language (XSL) é uma família de linguagem usadas para processar e transformar documentos XML. Essa família divide-se em três partes: XLS Transform (XSLT), XSL Formatting Objects (XSL-FO) e XML Path Language (XPATH). A primeira, como o nome indica, serve para aplicar transformações a ficheiros XML. A segunda é uma língua XML para especificar a formatação visual de um documento XML e, por fim, a terceira que é usada para navegar num documento XML. O O Node JS é um interpretador de JavaDcript assíncrono. E tem o objectivo de permitir a criação de aplicações de alta escalabilidade (por exemplo, servidor web), permitindo a gestão de milhares de conexões e eventos simultâneos numa única máquina física.

O trabalho é constítuido pelos ficheiros arq.xml ( dataset dos arqueossítios ), arq2html.xsl (regras de transformação de um arqueossítio xml para html), index.html (página índice dos arqueossítios), index.xsl (regras de geração de index.html), pr.xml (manifesto xml do presente trabalho), pr.html (manifesto html do presente trabalho), server.js (código do servidor), xml-split.xsl (regras para geração do dataset de cada um dos arqueossítios), e pela directoria dataset (ficheiros xml resultantes da aplicação de xml-split.xsl a arq.xml). Perfazendo no total um conjunto de 8 ficheiros e uma directoria.

No âmbito da unidade curricular de Desenvolvimento de Aplicações WEB (DAW) foi-nos proposto a resolução deste exercício (Trabalho para casa nº4), como forma de consolidar a matéria dada em aula. O exercício consistia em (1) gerar a directoria dataset a partir do documento arq.xml, (2) elaborar uma stylesheet geral para cada arqueossítio (arq2html.xsl), (3) construir o ficheiro servidor.js de modo a responder com o respectivo conteúdo de dataset/arq... e (4), se conseguisse, gerar um index, que permite a navegação para a página de cada arqueossítio.

O request do índice faz-se com "http://localhost:7777/0" e para os arqueossítios aplica-se o que foi feito em aula, ou seja, "http://localhost:7777/" + [1..122] .
