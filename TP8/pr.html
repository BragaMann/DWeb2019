Abstract:

No âmbito da unidade curricular de Desenvolvimento de Aplicações WEB (DAW) foi-nos proposto a resolução deste exercício (Trabalho para casa nº8), como forma de consolidar a matéria dada em aula. O exercício consistia em resolver o exercício 1 da Ficha dweb2019-e2 , desenvolvendo um servidor de páginas WEB em NodeJs, usando Express com uma ligação a uma base dados MongoDB. Até agora muito semelhante ao exercício anterior. Contudo, desta vez a arquitectura foi melhorada e o acesso à base de dados não é feito directamente pela View. Agora, os pedidos chegados da View são encaminhados para um router que por sua vez realiza os pedidos em que é necessário aceder à base de dados a um outro router, diminuindo assim o acoplamento e aumentado a segurança.

Após entrar na directoria TP8App executar npm start para correr o servidor, que atende na porta 3000.
