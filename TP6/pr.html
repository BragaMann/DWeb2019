Abstract:

No âmbito da unidade curricular de Desenvolvimento de Aplicações WEB (DAW) foi-nos proposto a resolução deste exercício (Trabalho para casa nº6), como forma de consolidar a matéria dada em aula. O exercício consistia em desenvolver um servidor de páginas WEB em NodeJs, usando Express. O servidor disponibiliza serviços CRUD (Create Read Update Delete) sobre o ficheiro Arquivo Sonoro de Ernesto Veiga de Oliveira.

Após entrar na directoria TP6App executar npm start para correr o servidor, que atende na porta 3000.
