Abstract:

O O Node JS é um interpretador de JavaScript assíncrono. E tem o objectivo de permitir a criação de aplicações de alta escalabilidade (por exemplo, servidor web), permitindo a gestão de milhares de conexões e eventos simultâneos numa única máquina física.

O trabalho é composto pela pasta stylesheets onde se encontram os ficheiros .css aplicam transformações à interface. Pelos ficheiros erro.pug, index.pug e layout.pug, que definem a estrutura html da página. O ficheiro gestaotarefas.js, onde está o código relativo ao servidor NodeJs. O ficheiro package-lock.json e o ficheiro tarefas.json onde neste serão guardados os dados, servindo de base de dados. E o pr.xml e pr.html, que são relativos ao manifesto deste trabalho.

No âmbito da unidade curricular de Desenvolvimento de Aplicações WEB (DAW) foi-nos proposto a resolução deste exercício (Trabalho para casa nº5), como forma de consolidar a matéria dada em aula. O exercício consistia desenhar uma simples página (única) de Gestão de Tarefas que possui as funcionalidades de listar e inserir tarefas. Para tal, tinha-se de definir todo o html para a formatação das páginas e um servidor que respondesse a pedidos. Nomeadamente a um GET da lista e a um POST de uma nova tarefa. Os dados relativos às tarefas serão guardados no ficheiro tarefas.json.
