Abstract:

O Extensible Stylesheet Language (XSL) é uma família de linguagem usadas para processar e transformar documentos XML. Essa família divide-se em três partes: XLS Transform (XSLT), XSL Formatting Objects (XSL-FO) e XML Path Language (XPATH). A primeira, como o nome indica, serve para aplicar transformações a ficheiros XML. A segunda é uma língua XML para especificar a formatação visual de um documento XML e, por fim, a terceira que é usada para navegar num documento XML.

O trabalho é constituído por um ficheiro pr.xml, pr.xsl e pr.html, todos relativos à transformação (.xsl) do conteúdo (.xml) numa página web (.html) de um Project Record (pr). Esta tarefa também é composta pelo ficheiro arq.xsl, usado para a transformação dos Arqueossítios do Nordeste Português ,conteúdo utilizado em aulas anteriores, num conjunto de páginas web, formando um website. Estas páginas encontram-se na pasta website que também faz parte da entrega. Perfazendo no total um conjunto de quatro ficheiros e uma directoria.

No âmbito da unidade curricular de Desenvolvimento de Aplicações WEB (DAW) foi-nos proposto a resolução deste conjunto de exercícios (Trabalho para casa nº3), como forma de consolidar a matéria relativa ao XSL dada em aula. Neste trabalho é dada uma ênfase acrescida ao XSLT, na medida em que é aplicado para transformar XML em html.
